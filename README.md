# UMKM-Tasikmalaya (Standalone)

Aplikasi katalog UMKM sederhana berbasis Next.js.

## Jalankan

```bash
npm install
npm run dev
```

Buka `http://localhost:7000`.

## Fitur

- Kategori produk (Celana / Celana Dalam / Singlet) + filter
- Katalog produk dengan varian ukuran S / M / L
- Keranjang dengan update qty dan total otomatis
- Checkout via WhatsApp ke `082338792512` (dinormalisasi otomatis ke format `62...`)
- Lokasi toko (embed + link maps)
- Lokasi pembeli via geolocation browser + fallback link manual
- Catatan ongkir: mengikuti aplikasi Gojek
